# array-uniq

> Create an array without duplicates

## Install

```
$ npm install array-uniq
```

## Usage

```js
import arrayUniq from 'array-uniq';

arrayUniq([1, 1, 2, 3, 3]);
//=> [1, 2, 3]

arrayUniq(['foo', 'foo', 'bar', 'foo']);
//=> ['foo', 'bar']
```
